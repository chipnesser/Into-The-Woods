# Hiking Vibes Generator

Static one-page trail picker for GitHub Pages.

## Publish fast

1. Create a new GitHub repo.
2. Upload the contents of this folder.
3. In GitHub, open `Settings` -> `Pages`.
4. Set the source to deploy from the main branch root.
5. Wait a minute or two for the site URL.

## Notes

- Weather loads live in the browser from Open-Meteo.
- Trail geocoding uses OpenStreetMap Nominatim in the browser.
- Drive times and trail lengths are starter estimates in `script.js`.
- Seasonal weighting is also configured in `script.js`.
